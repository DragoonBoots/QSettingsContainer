class dragoonboots::qsettingscontainer::QSettingsContainer¶

Settings container.

Use DGSETTINGS_SETTING, DGSETTINGS_SETTING_CAST, DGSETTINGS_STATE, and DGSETTINGS_STATE_CAST to define settings.

Get/set settings using the static methods defined after using the above macros.

DGSETTINGS_SETTING(T, name, default_val)¶

Create a new setting that persists between executions.

Defines three static methods: Get<name>, Set<name>, and GetDefault<name>.

Settings are stored persistently using QSettings; for non-persistent state information, see DGSETTINGS_STATE.

For enum-type variables with a different storage type, see DGSETTINGS_SETTING_CAST.

Example

DGSETTINGS_SETTING(int, Puppies, 0)
results in the following method definitions: 
static int GetPuppies();
static void SetPuppies(int value);
static int GetDefaultPuppies();

Parameters

  • T: value type; must be storable in a QVariant

  • name: Getter/setter name

  • default_val: Default value to use if no value has previously been stored; must be of type T

DGSETTINGS_SETTING_CAST(val_T, storage_T, name, default_val)¶

Similar to DGSETTINGS_SETTING, but uses different storage type and value types.

Example

DGSETTINGS_SETTING_CAST(Breed, unsigned int, PuppyBreed, Breed::Mix)
results in the following method definitions: 
static Breed GetPuppyBreed();
static void SetPuppyBreed(Breed value);
static Breed GetDefaultPuppyBreed();
casting the value to unsigned int when interfacing with QSettings.

Parameters

  • val_T: value type; must be castable using static_cast() to storage_t

  • storage_T: storage type; must be storable in a QVariant

  • name: Getter/setter name

  • default_val: Default value to use if no value has previously been stored; must be of type val_T

DGSETTINGS_STATE(T, name, default_val)¶

Create a new global state variable.

State information does not persist between executions. For persistent storage, see DGSETTINGS_SETTING.

For enum-type variables with a different storage type, see DGSETTINGS_STATE_CAST.

Example

DGSETTINGS_STATE(bool, DebugMode, false)
results in the following method definitions: 
static bool GetDebugMode();
static void SetDebugMode(bool value);
static bool GetDefaultDebugMode();

Parameters

  • T: value type; must be storable in a QVariant

  • name: Getter/setter name

  • default_val: Default value to use if no value has previously been stored; must be of type T

DGSETTINGS_STATE_CAST(val_T, storage_T, name, default_val)¶

Similar to DGSETTINGS_STATE, but uses different storage type and value types.

Example

DGSETTINGS_STATE_CAST(LogLevel, unsigned int, Logging, LogLevel::Warn)
results in the following method definitions: 
static LogLevel GetLogging();
static void SetLogging(LogLevel value);
static LogLevel GetDefaultLogging();
casting the value to unsigned int so it may be stored in memory.

Parameters

  • val_T: value type; must be castable using static_cast() to storage_t

  • storage_T: storage type; must be storable in a QVariant

  • name: Getter/setter name

  • default_val: Default value to use if no value has previously been stored; must be of type val_T
